# NETI — Java Network Programming Lab Trainer

so i was gonna study normally for the ICT lab test right? got bored as shit within 5 minutes. decided to vibe code an entire study game instead of actually studying. procrastination at its finest but hey, it works.

covers everything from the exam — socket programming, HTTP servers, REST, threading, networking fundamentals, architecture patterns — the whole thing. all 40 TODOs from the mock exam are in here too.

## what's in it

- **Concept Quiz** — MCQs with procedurally generated questions so you never get the same one twice
- **Code Completion** — fill in the exact exam TODOs, validates your answer
- **Pattern Matcher** — match concepts to definitions
- **Speed Round** — true/false with a timer, gets your heart rate up
- **Code Debug** — find the bug in broken Java code
- **Hard Mode** — unlocks at Level 5, red UI, half the time, double XP
- **Google Sign-in** — progress syncs across devices via Firebase
